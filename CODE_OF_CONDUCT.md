# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

These standards apply to all project spaces — issues, pull requests, discussions, commits, and any community channel where the project is represented.

## Enforcement

Concerns may be reported privately to the maintainers via the contact in [SECURITY.md](SECURITY.md). Reports are handled with discretion. Maintainers will review, decide on appropriate action, and respond within a reasonable timeframe.

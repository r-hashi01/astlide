# Astlide Docs

> **Note:** The documentation site is not yet built. This directory is the foundation for it.

## API Reference

TypeDoc generates the API reference from TSDoc comments in the source code.

```bash
# From repo root
bun run docs
# → generates docs/api/index.html
```

`docs/api/` is **git-ignored** (generated output). The source of truth is the TSDoc
comments in `packages/astlide/src/`.

## Structure (planned)

```
docs/
├── README.md          ← you are here
├── api/               ← generated by TypeDoc (git-ignored)
├── guides/            ← hand-written tutorials (future Starlight pages)
└── reference/         ← frontmatter / config reference (future Starlight pages)
```

## Workflow

| Action | Command |
|---|---|
| Regenerate API reference | `bun run docs` |
| Add a notable change to CHANGELOG | `bun run changeset` |
| Bump versions + generate CHANGELOG | `bun run version` |

# What is Astlide?

A presentation framework built on **Astro** — write slides in MDX.

- **MDX Support** — Markdown with component slots
- **Viewport scaling** — slides always fit any screen size
- **Theme system** — 7 built-in themes + CSS variable control
- **PDF export** — Playwright-based multi-page export
- **Keyboard navigation** — arrow keys, fullscreen, overview mode

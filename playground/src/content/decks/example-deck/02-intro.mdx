# What is Astlide?

A multi-deck presentation framework built on Astro.

- **MDX Support** — write slides in Markdown with component slots
- **Viewport scaling** — slides always fit any screen size
- **Theme system** — built-in themes + full CSS variable control
- **PDF / PNG export** — Playwright-based, multi-page merging with pdf-lib
- **Keyboard navigation** — arrow keys, fullscreen, overview mode
